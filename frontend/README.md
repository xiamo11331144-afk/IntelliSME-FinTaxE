# IntelliSME-FinTaxE 前端

AI 智能财税助手前端，基于 Vue 3 + Element Plus + RuoYi 框架。

## 技术栈

Vue 3.4 / Vite 5.3 / Element Plus 2.7 / Pinia 2.1 / ECharts / Axios

## 开发

```bash
npm install
npm run dev
```

默认端口 80，API 代理至 `http://localhost:8082`。

## 构建

```bash
npm run build:prod
```

## 环境变量

| 文件 | 用途 |
|---|---|
| `.env.development` | 开发环境 |
| `.env.production` | 生产环境 |
| `.env.staging` | 测试环境 |
